# Output Taxonomy

Pipeline outputs must be classified.

Candidate output:
- chunk id
- raw retrieval score
- retrieval source

Detection output:
- background mean
- background std
- evidence-SNR
- detection threshold

Selection output:
- selected atoms
- coverage gain
- redundancy penalty
- token cost

Generation output:
- answer
- cited atoms
- unsupported claims

Evaluation output:
- retrieval metrics
- answer metrics
- efficiency metrics

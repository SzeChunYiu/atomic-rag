# Root Cause Analysis

For every win or failure, answer:

- What changed from baseline?
- Which examples improved?
- Which examples degraded?
- Which atom caused the change?
- Was it detector, selector, reranker, or generator?
- Did token count change?
- Did evidence recall change?
- Did context precision change?
- Did source reliability change?

Root cause report format:
1. symptom
2. suspected cause
3. evidence
4. counterexample
5. fix or next experiment

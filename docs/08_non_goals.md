# Non-Goals

This project is not:
- a vague physics metaphor project
- a chatbot UI project
- a prompt-only RAG project
- a long-context stuffing project
- a benchmark cherry-picking exercise

Avoid:
- replacing all baselines too early
- claiming novelty before literature review
- optimizing on one dataset only
- ignoring latency and cost
- hiding failures

The goal is a measurable search method.

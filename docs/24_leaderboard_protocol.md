# Leaderboard Protocol

Do not chase leaderboards too early.

Phase 1:
Internal reproducibility on small public splits.

Phase 2:
Run broad datasets with frozen configs.

Phase 3:
Compare against strong published or open baselines.

Phase 4:
Submit or report only after sanity checks.

A result is meaningful only if it survives:
- multiple seeds
- multiple datasets
- held-out configs
- failure-case inspection

# Benchmarking and Evaluation Notes

Use retrieval and generation benchmarks separately.

Retrieval benchmarks:
- BEIR
- MS MARCO
- KILT provenance tasks

RAG answer benchmarks:
- Natural Questions
- HotpotQA
- MuSiQue
- 2WikiMultiHopQA
- LongBench

Evaluation tools:
- RAGAS-style faithfulness and context metrics
- exact-match and F1 where labels exist
- manual error analysis for failures

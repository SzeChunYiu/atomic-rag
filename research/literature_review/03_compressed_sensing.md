# Compressed Sensing Notes

Core idea:
Sparse signals can be reconstructed from fewer measurements under suitable assumptions.

RAG mapping:
- true answer evidence is sparse
- candidate chunks are noisy measurements
- selected evidence set is reconstructed signal
- token budget is measurement cost

Useful tools:
- L1 sparsity penalty
- coverage constraints
- incoherence or diversity constraints
- residual error tracking

Research action:
Find sparse recovery methods that can become practical selectors.

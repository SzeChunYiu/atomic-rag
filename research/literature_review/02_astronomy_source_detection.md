# Astronomy Source Detection Notes

Astronomy pipelines often include:
- background estimation
- background subtraction
- noise modeling
- threshold detection
- connected component detection
- deblending overlapping sources
- source measurement
- source catalog construction

RAG mapping:
- background = generic related text
- source = evidence atom
- detection = evidence discovery
- deblending = split overlapping claims
- catalog = cited evidence set

Potential novelty:
Apply this full pipeline to text retrieval, not just use SNR wording.

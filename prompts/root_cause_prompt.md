# Root Cause Prompt

Analyze this run result atomically.

Inputs:
- baseline artifacts
- Astro-CS-RAG artifacts
- metrics
- failure cases

Answer:
1. What improved?
2. What worsened?
3. Which atom caused the change?
4. Was the detector or selector responsible?
5. Was generation responsible?
6. What sanity checks passed or failed?
7. What experiment should run next?

Do not give vague explanations.
